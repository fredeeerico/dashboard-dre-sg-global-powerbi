# 00 - Sobre o Projeto

## Projeto

**Dashboard Financeiro para Leitura de DRE — SG Global Group**

Este projeto foi desenvolvido para consolidar e analisar dados financeiros da SG Global Group, com foco na construção de uma visão executiva da Demonstração do Resultado do Exercício (DRE).

A empresa possuía bases distintas para Brasil e Estados Unidos. Essas bases continham informações relevantes, mas estavam descentralizadas, com estruturas diferentes e problemas de qualidade que dificultavam a leitura analítica da performance financeira.

## Problema de negócio

O problema central era a ausência de uma visão consolidada e confiável da DRE.

Sem uma estrutura analítica única, a tomada de decisão ficava limitada por:

- Dados financeiros descentralizados
- Falta de padronização entre bases BR e US
- Inconsistências estruturais
- Problemas de encoding
- Datas ausentes
- Dificuldade de comparação entre países
- Baixa clareza sobre receita, despesa e resultado

## Solução desenvolvida

Foi construída uma solução de BI com Power BI, contemplando:

- Tratamento e consolidação das bases financeiras
- Criação de uma base oficial consolidada
- Modelagem analítica
- Medidas DAX para Receita, Despesa e Resultado
- Dashboard executivo com filtros e KPIs
- Visualizações para evolução temporal, países, receitas e despesas
